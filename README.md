# 💬 CloudChat - Django Chat App

A real-time chat application built using Django.

## 🚀 Features

- User Registration & Login
- Password Reset (OTP Email)
- Real-time Chat
- Search Users
- Edit/Delete Messages
- Export Chat

## 🛠 Tech Stack

- Django
- SQLite (Local) / PostgreSQL (AWS RDS)
- HTML, CSS, Bootstrap
- AWS (S3, CloudWatch)

## ⚙️ Run Locally

```bash
git clone https://github.com/morekarunanci/cloudchat_app_repo.git
cd cloudchat_app_repo
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
